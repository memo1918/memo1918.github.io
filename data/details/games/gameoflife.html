Conway's Game of Life is a famous cellular automaton. It consists of a grid of cells that can be either alive or dead. The game progresses in steps, where each cell's state is updated based on the states of its neighbors.

This project is a straightforward implementation of the Game of Life using Python and Pygame.


Core Functions

  • Conway's Game of Life, implemented according to the official rules.
  • Ability to start and stop the simulation in real-time.
  • Option to clear the grid and reset the simulation.


Motivation

Well, I was watching this great Conference made by Dylan Beattie called The Art of Code and they mentioned the Game of Life. I was blown away when I saw a simulation of the Game of Life running inside itself. It was fascinating to see how complex patterns could emerge from such simple rules.

Since the Game of Life is Turing complete, it can theoretically be used to compute anything. That idea stuck with me, and one night, out of curiosity (and boredom), I decided to code it myself.
