This project is a digital implementation of the Breakthrough board game, developed using Python and the Pygame library. The game features a basic AI opponent, allowing players to choose whether to play first or second.


Core Functions

  • Turn based game with AI as opponent.
  • Customizable window size, board size, and AI search depth.
  • Highlighted potential moves when selecting a piece.
  • Win and lose conditions and ability to restart the game.


Technologies and Concepts

While Python is versatile and easy to use, Pygame provides essential functionalities such as window creation, input handling, and rendering. This makes them a perfect choice for developing simple 2D games.

The game follows OOP principles, with separate classes for core elements like the 'Board', 'Squares', 'Pieces', and 'AI'. This structured approach makes the code highly reusableâ€”to the point that I have used this game as a template for developing other games.

The AI is implemented using the Minimax algorithm with a heuristic function to evaluate board states, enabling it to make strategic decisions.


Development

I started making the game as for two person to play withouth AI. However, I always kept AI integration in mind and ensured that the game logic was flexible enough to accommodate it.

The first version of the AI used a simple one-step lookahead approach, evaluating only the current board state to decide on a move.

Implementing an N-step lookahead required the Minimax algorithm, which was challenging on its own. Fine-tuning the heuristic function added another layer of complexity, and it took two full revisions to finalize a working AI that could make strategic decisions.


Potential Future Plans

Currently, increasing the board size or AI search depth significantly slows down decision-making.

A potential solution is to implement optimization techniques such as Alpha-Beta Pruning to improve the efficiency of the Minimax algorithm.

Additionally, integrating different AI algorithms could provide a comparison of different algorithms.
