Cloudseeker is a project that aims to make it easier to compare cloud providers and their services. Major cloud providers like GCP, Azure, and AWS each have their own pricing models, making comparisons difficult. Our platform brings them all together in one place, allowing users to quickly identify the most cost-effective cloud services for their needs.
Cloudseeker includes fetching up-to-date data from APIs, easy filtering options and an easy-to-use comparison page where you can see the price differences.


Core Functions

The web app provides basic filtering and search capabilities also being able to browse through different categories (VMs, Storage etc.). You can add instances to "basket" then to compare the prices between different providers.

Instances are mapped based on their initial configurations, with pricing and specifications fetched from APIs and stored in a database.
